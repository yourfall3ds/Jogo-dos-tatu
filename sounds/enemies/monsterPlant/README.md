# Sons — Planta Monstro

Coloque os arquivos de som da Planta Monstro aqui. Formatos suportados: `.ogg`, `.mp3`, `.wav`

| Arquivo       | Quando toca                                 |
|---------------|---------------------------------------------|
| `hop.ogg`     | Ao pular (cada hop)                         |
| `damage.ogg`  | Ao tomar dano de um tiro                    |
| `death.ogg`   | Ao morrer                                   |
| `attack.ogg`  | Ao atacar o jogador em contato              |

> **Dica**: Você também pode fazer upload dos sons pelo painel admin em `http://localhost:5500/admin.html`.
> Os sons carregados pelo admin ficam salvos permanentemente no navegador (IndexedDB) e têm
> prioridade sobre os arquivos desta pasta.
