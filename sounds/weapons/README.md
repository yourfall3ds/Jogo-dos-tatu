# Sons das Armas

Coloque os arquivos de som das armas aqui. Formatos suportados: `.ogg`, `.mp3`, `.wav`

| Arquivo       | Quando toca                                 |
|---------------|---------------------------------------------|
| `fire.ogg`    | Ao disparar um tiro                         |
| `reload.ogg`  | Ao iniciar uma recarga                      |
| `empty.ogg`   | Ao tentar atirar sem munição                |

> **Dica**: Você também pode fazer upload dos sons pelo painel admin em `http://localhost:5500/admin.html`.
