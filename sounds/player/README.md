# Sons do Jogador

Coloque os arquivos de som do jogador aqui. Formatos suportados: `.ogg`, `.mp3`, `.wav`

| Arquivo       | Quando toca                                 |
|---------------|---------------------------------------------|
| `jump.ogg`    | Ao pular no chão                            |
| `land.ogg`    | Ao pousar após uma queda                    |
| `walljump.ogg`| Ao executar um wall jump                    |
| `damage.ogg`  | Ao tomar dano                               |
| `death.ogg`   | Ao morrer                                   |

> **Dica**: Você também pode fazer upload dos sons pelo painel admin em `http://localhost:5500/admin.html`.
> Os sons carregados pelo admin ficam salvos permanentemente no navegador (IndexedDB).
